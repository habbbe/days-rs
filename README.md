# Days

A tool to count the days (and time) between two dates

## Build

`cargo build --release`

## Usage
```
Usage: days [DATE|now|today|tomorrow|...] until DATE|now|today|tomorrow|...

Examples:
    days until 25 dec
    days until next 25 dec
    days 1 January 1970 until now
    days until 23 may 2030
```
